1. 決定長度 [[Wing 槳長度]]: 裁 5cm, 一邊 2.5
2. 切割
	1. A 用鋸子 (fine-tooth hacksaw, 24–32 TPI, 越細越好) 鋸, 慢慢鋸不要用力壓避免纖維散開, 也可以先用膠帶 (masking tape 或 electrical tape) 纏繞避免纖維散開.
	   B. 用 **pipe cutting jig / miter box**
	2. 切碳纖之前在鋸片上噴一點水或 WD-40
	3. 用砂紙磨, 可以先用膠帶纏繞避免纖維散開
	4. 清理碳纖粉, 碳纖粉不要吸進去。用濕紙巾擦或戴簡單口罩
3. 粗糙化, 把 shaft 跟blade重疊的部份用砂紙磨, 之後會黏得比較緊, shaft 插入 blade 之後可以看出範圍

4. 對齊
	1. Shaft 裝好調到零度![[Pasted image 20260308093306.png]]
	2. 在blakde 16cm 處貼膠帶![[Pasted image 20260308093205.png]]
	3. 找東西靠上去![[Pasted image 20260308093350.png]]
	4. 往下壓讓他們轉齊![[Pasted image 20260308093420.png]]
5. 黏
	1. 熱風槍加熱 shaft 到50c/122f![[Pasted image 20260308092328.png]]
	2. 用hot glue gun 在 shaft 上膠![[Pasted image 20260308092552.png]]
	3. 拿金屬片把膠抹平![[Pasted image 20260308092635.png]]
	4. 重新加熱一次(最高60c)![[Pasted image 20260308092659.png]]
	5. 加熱 blade ![[Pasted image 20260308092802.png]]
	6. Blade 內壁加熱![[Pasted image 20260308092831.png]]
	7. 組裝, 在膠還沒完全冷之前輕輕左右轉 1 mm 讓膠均勻再對齊 mark。 ![[Pasted image 20260308092905.png]]
	8. 切膠
		1. 塗酒精![[Pasted image 20260308093735.png]]
		2. 用刀子沿著接合縫切 (和 shaft 垂直)![[Pasted image 20260308093835.png]]
